# Dependencies

1- React 16.13   <br/>
2- Mongodb  <br/>
3- React  <br/>
4- Redux  <br/>

# To Start Application

1- Install MongoDB Compass <br/>
2- Run backend to type npm install and  npm start in root folder  <br/>
3- Run frontend to type yarn install and yarn start in ./frontend folder  <br/>
